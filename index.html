A solução

PIX orquestrado.
3 passos. 5 segundos.

A API Flowt automatiza toda a orquestração — deep link inteligente, auto-copy e redirecionamento direto para a tela de PIX do banco escolhido.

⚡
Deep Link Inteligente
Abre o app do banco direto na tela de PIX Copia e Cola, sem navegação manual. Suporte a Nubank, Itaú, Bradesco, BB, Inter, C6 e mais.
📋
Auto-Copy automático
O código PIX é copiado para o clipboard no momento exato do clique. O banco detecta e oferece colar automaticamente.
🔁
Preferência de banco salva
Na segunda compra, o fluxo é ainda mais rápido — o banco preferido do cliente já está selecionado.
🔌
Integração simples
API REST com uma única chamada POST. Funciona sobre qualquer PSP existente — sem trocar de gateway.
📊
Métricas em tempo real
Dashboard com taxa de conversão, tempo médio de checkout e abandono por etapa.
🛡
Sem intermediar pagamentos
A Flowt é camada de UX, não processadora. O dinheiro flui pelo seu PSP atual. Zero compliance extra.
